# Sistem Manajemen Toko (Point of Sale)

Aplikasi berbasis web sederhana namun fungsional untuk mengelola stok barang, transaksi kasir, hingga laporan keuangan bulanan. Proyek ini dibangun menggunakan Python dengan Framework Flask.
